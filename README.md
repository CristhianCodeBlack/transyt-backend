# 🚛 TRANSYT - Plataforma de Capacitación para Conductores

## 📋 Descripción
Sistema integral de capacitación empresarial diseñado específicamente para conductores profesionales. Incluye gestión de cursos, evaluaciones, certificados y capacitaciones en vivo con Microsoft Teams.

## 🚀 Características Principales
- **📱 PWA** - Instalable como app nativa en móviles
- **🔄 Modo Offline** - Cursos disponibles sin conexión
- **🎓 Certificados Digitales** - Generación automática con códigos QR
- **📹 Capacitaciones en Vivo** - Integración con Microsoft Teams
- **📊 Dashboard Completo** - Seguimiento de progreso en tiempo real
- **🔔 Notificaciones Push** - Recordatorios de cursos y evaluaciones

## 🛠️ Tecnologías
### Backend
- **Java 17** + Spring Boot 3.x
- **PostgreSQL** - Base de datos
- **JWT** - Autenticación
- **Cloudinary** - Almacenamiento de archivos
- **Microsoft Graph API** - Integración Teams

### Frontend
- **React 18** + Vite
- **Tailwind CSS** - Diseño responsive
- **Service Worker** - PWA y offline
- **React Router** - Navegación SPA

## 🚀 Despliegue
- **Backend**: Render.com
- **Frontend**: Render.com
- **Base de datos**: PostgreSQL en Render

## 👥 Roles de Usuario
- **ADMIN** - Gestión completa del sistema
- **INSTRUCTOR** - Creación y seguimiento de cursos
- **EMPLEADO** - Acceso a cursos y evaluaciones

## 📱 Optimizado para Conductores
- Interfaz móvil intuitiva
- Navegación con botones grandes
- Modo offline para rutas sin conexión
- Notificaciones de recordatorio
- Acceso rápido desde pantalla de inicio
